Sensitive Govermental Documents

Government departments, such as the Ministry of Defence, produce a large number of written communications essential for the international operations of government. Freedom of Information Laws legislate that these documents should be released to the public.

Sensitive Information

However, due to the sensitive nature of these documents, each document needs to be reviewed to identify and protect any sensitivities.

Classifiers

This website helps users classify documents using three different classifiers. The first classifier is a linear model (logistic regression); the second classifier is a tree-based model (xgboost); and finally, the third model is a deep learning model (long short-term memory).

Classifier Explanations

Furthermore, this website interprets and explains each classifier's predictions and visualises them using third-party libraries: LIME, SHAP, and ELI5.

View these document explanations in real time.
